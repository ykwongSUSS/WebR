flexTable

flextable takes a data.frame and returns a flextable. flextable also allows you to merge cells, add header rows, add footer rows, change all formatting and specify how data should be displayed in cells. Table contents can also contain a mix of content types such as text and images.

Library

library(tidyverse)
library(flextable)
