table

  1. table works on dataframes and tibbles.
  2. used for counting categories into a frequency table
  3. addmargins function works for matrix and table forms.
library(tidyverse)

1 Data - table (tbl)

  1. table function is in baseR and creates a contingency table from a matrix form.
  2. addmargins works for matrix and table forms

1.1 Matrix form

# Table 2.2 of Agresti
M <- matrix (c(189,10845,104,10933), nrow=2, byrow=TRUE,
             dimnames = list(group=c("Placebo","Aspirin"),
                                c("yes","no")))
class(M)
[1] "matrix" "array" 
addmargins(M)
         
group     yes    no   Sum
  Placebo 189 10845 11034
  Aspirin 104 10933 11037
  Sum     293 21778 22071

1.2 Table form

tbl3 <- as.table(M)
class(tbl3)
[1] "table"
addmargins(tbl3,2)
         
group       yes    no   Sum
  Placebo   189 10845 11034
  Aspirin   104 10933 11037

1.3 Proportions table

# margin=1 proportionates the rowSums
ptbl <- prop.table(tbl3, margin=1) |> round(4)
ptbl

  Placebo 0.0171 0.9829
  Aspirin 0.0094 0.9906
